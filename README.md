# Python-Based-Students-Audit-Automated-Reporting
I noticed that identifying students for intervention took up valuable planning time. I applied my technical literacy in Python to automate this workflow, ensuring no student's progress is overlooked due to manual oversight.
# The Problem
In a busy classroom of 20+ students, manually checking every single grade to see who needs extra help is time-consuming and prone to human error. I wanted to create a solution that gives teachers their time back.

# The Solution
I built this Python-based tool to handle the "boring" part of grading. Instead of looking at a spreadsheet line-by-line, a teacher can simply run this script to get an instant, clear report of which students are on track and which ones need a little more support.

# What’s Inside?
- Smart Detection: Automatically finds the "Grade" column, even if the file is a bit messy.
- Instant Reporting: Generates a new, clean list labeled "Needs Support" or "On Track."
- Efficiency: Processes a whole class of students in less than a second.

**Impact**
By automating this administrative task, educators can focus less on data entry and more on what matters: supporting their students.
